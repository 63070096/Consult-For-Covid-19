โครงาน: Application ให้การปรึกษาเกี่ยวกับ Covid-19

ผู้จัดทำ: นายบารเมษฐ สายจันยูร 63070096, นายศุภวัฒน์ อุสาห์นนท์ 63070171, นายสุเมธ เขตขามม 63070178

อาจารย์ที่ปรึกษา: รศ.ดร ปานวิทย์ ธุวะนุติ

สถานศึกษา: สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง

เนื่องในปัจจุบันเกิดวิกฤติการระบาดของเชื้อ Coronavirus หรือ Covid-19 ซึ่งมีผู้ติดเชื้อและผู้เสียชีวิตเป็นจำนวนมาก ทั่วโลกรวมถึงประเทศไทยที่สถานการตอนนี้ที่ยังไม่มีวี่แววว่าจะคลายลงอีกทั้งวัคซีนก็ยังไม่สามารถเข้าถึงได้ทุกรายทำให้มีผู้คน จำนวนไม่น้อยเกิดความวิตกกังวลว่าตนเองจะได้รับเชื้อไปแล้วทั้งๆที่ไม่ได้อยู่ในกลุ่มเสี่ยง คณะผู้จัดทำจึงได้ทำกาารออกแบบ Application นี้ขึ้นมาเพื่อให้คำปรึกษาแก่ผู้ใช้ว่าคุณควรไปพบแพทย์เพื่อรับการตรวจ หาเชื้อ Covid-19 หรือไม่โดยตัว Application นี้จะใช้ภาษา C เป็นหลักโดยอัลกอริทึ่มที่ไม่ซับซ้อนซึ่งมีส่วนของ Structure ในการเก็บข้อมูลของผู้ใช้และ Condition if-else เข้ามาช่วยในการซักถาม การทำ Application ด้วยภาษา C นั้นจะเป็น ในรูปแบบของ Console Application จากการทำ Application นี้มีข้อเสนอว่า Source Code นี้ควรนำไปพัฒนาต่อโดยภาษาอื่นๆเชื่น C++ C# หรือ Python ซึ่ง ภาษาเหล่านี้สามารถนำไปต่อยอดทำ GUI ได้ เพื่อเพิ่มความน่าสนใจและความน่าเชื่อถือของ Application มากยิ่งขึ้น

Application นี้ได้รับการพัฒนาด้วยภาษา C ด้วยฟังก์ชั่นต่างและ Condition If-Else และการนำ console i/o มาช่วยในการประมวลผลบน Console App โดยเบื้องต้นเมื่อกดทำการกด Start Survey แล้วผู้ใช้จะต้องทำการกรอก ชื่อ นามสกุล อายุ จังหวัด และ อำเภอ เมื่อเสร็จแล้วระบบจะมีคำถามให้ด้วยกัน 9 คำถาม เป็น YES/NO Question แต่ละคำถามจะให้คะแนนที่แตกต่างกันไปตามความเสี่ยงดังนี้

1.ก่อนหน้านี้คุณเคยเดินางไปยังพื้นที่เสี่ยงหรือไม่ (20 คะแนน)

2.คุณเคยพูดคุยหรือใกล้ชิดกับผู้ที่มาจากพื้นที่เสี่ยงหรือไม่ (20 คะแนน)

3.คุณเคยพูดคุยหรือใกล้ชิดกับผู้ป่วยยืนยันหรือไม่ (40 คะแนน)

4.คุณเคยอยู่สถานที่ปิดหรืออยู่ในเหตุการณ์ที่มีผู้คนมากกว่า 100 คนหรือไม่ (30 คะแนน)

5.คุณมีไข้มากกว่า 37.5 องศาหรือไม่ (40 คะแนน)

6.มีอากการไอหรือไม่ (40 คะแนน)

7.คัดจมูกหรือไม่ (40 คะแนน)

8.รู้สึกหายใจลำบากหรือไม่ (50 คะแนน)

9.ไม่สามารถรับรู้กลิ่นได้ (50 คะแนน)
